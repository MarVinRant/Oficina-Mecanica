# 🚗 Oficina Mecânica & Auto Center - Landing Page

## 📖 Sobre o Projeto
Uma landing page robusta, direta e responsiva, desenvolvida para oficinas mecânicas, auto centers e centros de estética automotiva. O principal objetivo deste projeto é transmitir confiança e credibilidade aos motoristas, facilitando o contato rápido para orçamentos e agendamentos de revisões.

Este projeto integra o meu portfólio profissional de desenvolvimento Front-End, destacando minha habilidade em criar interfaces voltadas para conversão e prestação de serviços no comércio local.

## ✨ Principais Funcionalidades
* **Design Focado em Credibilidade:** Layout estruturado com cores e elementos que passam a sensação de profissionalismo, transparência e segurança – valores essenciais para o setor automotivo.
* **Mobile-First & Responsividade Total:** Como muitos clientes buscam por oficinas pelo celular (muitas vezes em situações de emergência), o site foi otimizado para navegação rápida e clara em qualquer tamanho de tela.
* **Catálogo de Serviços:** Seção dedicada para apresentar os principais serviços oferecidos (ex: Revisão Geral, Troca de Óleo, Suspensão, Freios, etc.), facilitando a identificação da solução pelo cliente.
* **Atendimento Ágil (CTA):** Botões de "Solicitar Orçamento" estrategicamente posicionados, direcionando o cliente imediatamente para o WhatsApp para um contato sem fricção.
* **Performance e Velocidade:** Código limpo e otimizado para garantir carregamento instantâneo, essencial para reter usuários que buscam soluções rápidas.

---

## 🛠️ Tecnologias Utilizadas
A landing page foi desenvolvida utilizando as linguagens essenciais da web "na unha", garantindo máxima performance e total controle sobre a interface, sem o peso de frameworks externos:

* **HTML5:** Estrutura semântica para excelente indexação (SEO local), ajudando a oficina a ser encontrada facilmente na região.
* **CSS3:** Estilização moderna, uso de Grid/Flexbox para organização dos serviços e design perfeitamente adaptável a dispositivos móveis.
* **JavaScript (Vanilla):** Implementação de lógicas de interface e interatividade fluida para o usuário.
